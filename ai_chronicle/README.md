## Local Development

1. Install dependencies:
   ```bash
   cd ai_chronicle
   npm install
   ```

2. Create a `.env` file with your Anthropic API key:
   ```
   ANTHROPIC_API_KEY=your_api_key_here
   ```

3. Install Vercel CLI (if not already installed):
   ```bash
   npm install -g vercel
   ```

4. Run development server:
   ```bash
   npm run dev
   # or
   vercel dev
   ```

5. Open `http://localhost:3000` in your browser

## Deployment to Vercel

1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```

2. Login to Vercel:
   ```bash
   vercel login
   ```

3. Deploy:
   ```bash
   vercel --prod
   ```

4. Set environment variable in Vercel dashboard:
   - Go to your project settings
   - Add `ANTHROPIC_API_KEY` as an environment variable
   - Redeploy if necessary

## How It Works

1. **Story Generation**: The game communicates with Claude AI through serverless API endpoints
2. **State Management**: Game state (protagonist, companions, story history) is maintained client-side
3. **Dice Rolls**: When the AI determines a choice is risky, it triggers a dice roll animation
4. **Narrative Flow**: The AI generates coherent, continuous narrative that respects past choices
5. **Death Detection**: The AI determines when the protagonist dies and triggers the game over sequence

## Game Mechanics

- **Choices**: Occasionally, the AI presents meaningful choices that affect the story
- **Dice Rolls**: Dramatic actions trigger a dice roll with success/failure outcomes
- **Companions**: NPCs can join your journey and persist throughout the story
- **Death**: If your protagonist dies (from poor choices or failed rolls), the game ends with a fade to black

## Customization

### Changing the Theme

Edit `public/styles.css` to modify colors, fonts, and styling. The main color variables are defined in `:root`:

```css
:root {
    --sepia-dark: #3e2723;
    --sepia-medium: #704214;
    --sepia-light: #8d6e63;
    --sepia-page: #f4e8d8;
}
```

### Modifying Story Generation

Edit `api/story.js` to change:
- `SYSTEM_PROMPT`: Adjust the AI's storytelling style and rules
- `model`: Change the Claude model version
- `temperature`: Adjust creativity (0.0-1.0, higher = more creative)
